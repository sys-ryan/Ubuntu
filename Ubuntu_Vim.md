### set environment
`vi ~/.vimrc`

```
set number          
set tabstop=4         
set ignorecase      
set hlsearch         
set fileencodings=utf-8,euc-kr    
set bs=indent,eol,start    
set ruler              
set title               
set showmatch   
set nowrap        
set wmnu          
syntax on    
```
